Compiling from Source

To compile Pixelorama's source code, all you have to do is download Godot 3.1. You don't need the mono version since Pixelorama is written entirely with GDScript. Then, you clone Pixelorama from its GitHub repository, load the project within Godot, and you run it! Simple as that!
